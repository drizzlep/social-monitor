# 良禽佳木·社媒情报平台 — 项目规格书

> 2026-03-24 by 田瓜

---

## 核心定位

**目标：** 监控中国社交媒体（微信公众号、小红书为主）上实木/原木定制家具行业的品牌声量、用户真实反馈、竞品动态。
**性质：** 自用情报工具，非 SaaS 产品
**用户：** 米田（良禽佳木团队）

---

## 数据范围

### 监控平台（优先级排序）

| 优先级 | 平台 | 说明 |
|--------|------|------|
| P0 | 微信公众号 | 品牌 announcements + 深度文章 |
| P0 | 小红书 | 种草内容 + 用户口碑 |
| P2 | 抖音/视频号 | 视频内容，后期接入 |

### 关键词体系（三类）

#### 品牌词（竞品监控）
```
木墨、朴愫木作、上屹木作、明时木作、奇才木作、兰宝、百强、源氏木语
```

#### 行业词（趋势监控）
```
全屋定制、实木定制、原木定制、全屋整装、高端全屋定制、实木家具、
原木家具、柜墙门一体化、轻高定
```

#### 需求词（用户声音）
```
实木定制 哪家好、全屋定制 甲醛/环保、实木定制 攻略/避坑、
原木定制 价格/多少钱、实木vs定制哪个好、定制家具 推荐/口碑
```

## 数据维度

每个抓取的内容需要包含：

| 字段 | 说明 | 数据来源 |
|------|------|---------|
| 标题 | 文章/帖子标题 | 平台 |
| 平台 | 微信公众号 / 小红书 | 系统字段 |
| 作者/账号 | 发布者 | 平台 |
| 发布时间 | 年月日时分 | 平台 |
| 原文链接 | 可追溯 | 平台 |
| 摘要 | AI 生成简要 | 平台内容 |
| 点赞/在看数 | 传播广度 | 平台 |
| 评论数 | 用户真实声音 | 平台 |
| 收藏数 | 价值认可度 | 小红书 |
| 关键词匹配 | 触发该内容的关键词 | 关键词体系 |
| 品牌归属 | 属于哪个竞品 | 品牌词匹配 |

## 更新频率

- **Day 1-7（验证期）：** 每天一次
- **跑顺之后：** 每三天一次
- **稳定后：** 每周一次

## 技术方案

### 存储方案
- 数据格式：JSON，存于 `~/.openclaw/workspace/social-monitor/data/`
- 每条平台数据单独文件，按日期归档
- 网页展示：Vercel 托管（复用的经验）

### 数据来源
- 微信公众号：Agent-Reach 工具
- 小红书：Agent-Reach 工具或百度搜索 API
- 互动数据：通过真实抓取获取，不编造

### 扩展计划
- 第二期：抖音/视频号接入
- 第二期：钉钉多维表格同步
- 第二期：竞品对比看板

---

## 验收标准

1. 数据真实性 100%：所有数据必须来自真实搜索/抓取，禁止编造
2. 关键词覆盖：三套词全部可用
3. 互动数据可抓：点赞/评论/收藏优先
4. 网页可访问：Vercel 部署，有列表页
5. 可排序：按时间/互动量/品牌 排序
